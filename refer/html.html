속성 설명
<a> <a> 태그는 다른 페이지 이동을 설정합니다.
<area> 이미지 맵(image-map)에서 하이퍼링크가 연결될 영역을 정의할 때 사용합니다.
<article> <article> 태그는 해당 문서나 페이지 또는 사이트와는 완전히 독립적으로 구성할 수 있는 요소를 정의할 때 사용합니다.
<aside> 문서의 주요 내용과 간접적으로만 연관된 부분을 나타냅니다. 주로 사이드바 혹은 콜아웃 박스로 표현합니다.
<body> <body> 태그는 문서 본문에 해당하는 콘텐츠를 포함하는 영역입니다.
<br> <br> 태그는 텍스트를 줄바꿈 할 때 설정합니다.
<button> <button> 태그는 버튼을 만들 때 설정합니다.
<caption> <caption> 태그는 table 요소의 제목을 설정합니다.
<center> <center> 태그는 HTML5에서 지원되지 않고 대신 CSS에 사용합니다.
<code> <code> 태그는 코드를 나타내는 텍스트를 설정합니다.
<col> <col> 태그는 <colgroup> 요소에 속하는 열의 속성을 설정합니다.
<colgroup> <colgroup> 태그는 표의 열을 묶는 그룹을 설정합니다.
<div> <div> 태그는 문서의 영역을 나누거나 그룹화할 때 설정합니다.
<em> <em> 태그는 텍스트를 강조할 때 설정합니다.
<fieldset> 요소에서 연관된 요소들을 하나의 그룹으로 묶을 때 사용합니다.
<figcaption> <figcaption> 태그는 이미지나 다이어그램, 사진과 같이 문서의 주요 흐름과는 독립적인 콘텐츠를 정의하는 <figure> 요소의 캡션(caption)을 정의할 때 사용합니다.
<figure> figure 는 보통 이미지, 영상 등을 올릴 때 사용하는 태그입니다.
<font> 텍스트의 폰트(font), 색상 그리고 크기를 정의할 때 사용합니다.
<footer> footer 요소는 HTML 문서나 섹션(section) 부분에 대한 푸터(footer)을 정의합니다.
<form> 웹 페이지에서의 입력 양식을 의미합니다.
<frame> 요소에 포함되는 하나의 프레임(frame)을 정의할 때 사용합니다.
<h1> ~ <h6> <h1> ~ <h6> 태그는 섹션의 제목을 설정합니다.
<head> <head> 태그는 제목과 스크립트, 스타일 시트, 메타 정보 등을 제공하는 영역입니다.
<header> 태그는 문서나 특정 섹션(section)의 헤더(header)를 정의할 때 사용합니다.
<hr> <hr> 태그는 요소간에 주제별 구분을 설정합니다.
<html> <html> 태그는 웹 문서의 최상위 요소 문서의 루트를 나타냅니다.
<i> 글자를 기울여서 표시하는 태그로, italic의 약자입니다.
<iframe> HTML문서의 안에 또 다른 HTML문서를 삽입할 수 있도록 해주는 기능을 하는 태그이다.
<img> <img> 태그는 웹문서에서 이미지를 설정합니다.
<input> <input> 태그는 사용자 입력정보를 받을 때 설정합니다.
<label> 사용자 인터페이스(UI) 요소의 라벨(label)을 정의할 때 사용합니다. 또한 label태그의 for 속성은 라벨(label)과 결합될 요소를 명시합니다.
<legend> fieldset 태그로 묶은 요소들에 대한 제목이나 설명을 붙일 때 사용.
<li> <li> 태그는 목록의 항목을 설정합니다.
<link> lt;link> 태그는 외부 파일을 연결할 때 설정합니다.
<main> <main> 태그는 웹 문서의 주요 콘텐츠 영역을 설정합니다.
<map> 이미지맵 태그(map, area 태그) map 태그는 이미지맵을 만드는 태그입니다.
<meta> 메타태그는 웹 서버와 웹 브라우저간에 상호 교환되는 정보를 정의하는데 사용합니다.
<nav>
태그는 다른 페이지 또는 현재 페이지의 다른 부분과 연결되는 네비게이션 링크(navigation links)들의 집합을 정의할 때 사용합니다.
<noscript> <noscript> 태그는 클라이언트 사이드 스크립트(client-side scripts)를 사용하지 않도록 설정했거나, 스크립트를 지원하지 않는 브라우저를 위한 별도의 콘텐츠를 정의할 때 사용합니다.
<ol> ol태그는 ordered list(순서가 정해진 목록)의 약자입니다.
<option> 태그의 value 속성은 해당 옵션이 선택될 때 서버로 제출되는 값을 명시합니다.
<p> <p> 태그는 단락을 나타낼 때 설정합니다.
<pre> <pre> 태그는 줄바꿈과 공백이 적용되는 단락을 정의합니다.
<script> 스크립트를 작성할 수 있게 설정합니다.
<section> 주제별 영역들을 그룹화 하기위해 사용한다.
<select> select 태그는 말 그대로 다양한 선택지 중에서 선택을 하는 드롭박스를 만드는 태그입니다.
<span> HTML 문서에서 인라인 요소(inline-element)들을 하나로 묶을 때 사용합니다.
<strong> <strong> 태그는 텍스트의 중요성을 강조할 때 설정합니다.
<style> <style> 태그는 웹 문서에 스타일 정보를 설정합니다.
<svg> SVG태그는 Scalable Vector Graphics의 약자로 벡터 기반 그래픽을 XML 형식으로 정의하는 것을 의미합니다.
<table> <table> 태그는 표를 설정합니다.
<tbody> <tbody> 태그는 표의 본문 영역을 그룹화합니다.
<td> <td> 태그는 표의 열(셀)을 설정합니다.
<tfoot> <tfoot> 태그는 표의 푸터 영역을 그룹화합니다.
<th> <th> 태그는 행이나 열에 타이틀을 설정합니다.
<thead> <thead> 태근 표의 헤더 영역을 그룹화합니다.
<title> <title> 태그는 브라우저의 제목 또는 탭에 표시되는 문서의 제목을 설정합니다.
<u> underline이라는 영어단어의 앞글자 1개에서 따온 태그로, 웹페이지의 텍스트에 단순히 밑줄을 표시하는 경우에 사용하는 태그입니다.
<ul> 순서가 없는 HTML 리스트(list)를 정의할 때 사용합니다.
<video> 비디오나 영화를 삽입할 때 사용합니다.
