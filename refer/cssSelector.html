기본 선택자

속성 예시 설명
태그 선택자 p {color: #fff} p 태그를 선택하여 글씨 색을 하얀색으로 변경합니다.
클래스 선택자 .class {color: #fff} class 클래스를 선택하여 글씨 색을 하얀색으로 변경합니다.
아이디 선택자 #id {color: #fff} id 아이디를 선택하여 글씨 색을 하얀색으로 변경합니다.
그룹 선택자 div, p {color: #fff} div 태그와 p태그를 선택하여 글씨 색을 하얀색으로 변경합니다.
전체 선택자 * {color: #fff} 전체태그를 선택하여 글씨 색을 하얀색으로 변경합니다.

계층선택자

속성 예시 설명
자식 선택자
(child selector)		 div > p {color: #fff} div > p 자식요소 p만 선택하여 적용
형제 선택자
(child selector)		 div > p {color: #fff} div > p 선택자1 바로 다음에 오는 동생 선택자2를 선택
속성 선택자
(child selector)		 div > p {color: #fff} p 태그를 선택하여 글씨 색을 하얀색으로 변경합니다.

속성 선택자

종류 예시 설명
[name] a[href] a 태그 속성 중에 href가 있으면 선택합니다.
[name="value"] a[href="#header"] a 태그 href 속성의 값이 '#header'이면 선택합니다.
[name~="value"] a[href~="web"] a 태그 href 속성의 값이 'web'을 포함한 요소를 선택합니다.
[name|="value"] a[href|="web"] a 태그 href 속성의 값이 'web'이거나 'web'으로 시작하는 요소를 선택합니다.
[name^="value"] a[href^="web"] a 태그 href 속성의 값이 'web'으로 시작하는 태그를 선택합니다.
[name*="value"] a[href*="web"] a 태그 href 속성의 값이 'web'이 포함되어 있는 태그를 선택합니다.
[name$="value"] a[href$="com"] a 태그 href 속성의 값이 'com'으로 끝나는 태그를 선택합니다.

가상 클래스

종류 예시 설명
:first-child li:first-child {color: #fff} 형제 요소 중 제일 첫 요소의 색을 흰색으로 나타냅니다.
:last-child li:last-child {color: #fff} 형제 요소 중 제일 마지막 요소의 색을 흰색으로 나타냅니다.
:first-of-type p:first-of-type {color: #fff} 형제 요소 중 자신의 유형과 일치하는 제일 첫 요소의 색을 흰색으로 나타냅니다.
:last-of-type p:last-of-type {color: #fff} 요소 중에서 맨 마지막으로 등장하는 특정 타입의 요소를 모두 선택합니다.
:nth-child(2) li:nth-child(2) {color: #fff} li목록의 두 번째 선택자의 칼라를 흰색으로 나타냅니다.
:nth-last-child(2) li:nth-last-child(2) {color: #fff} li목록의 마지막 두 번째 선택자의 칼라를 흰색으로 나타냅니다.
:nth-of-type(2) p:nth-of-type(2) {color: #fff} 두번째 p선택자에게 칼라를 흰색으로 나타냅니다.
:nth-last-of-type(2) p:nth-last-of-type(2) {color: #fff} 마지막 두 번째 p 선택자의 칼라를 흰색으로 나타냅니다.
:only-child p:only-child {color: #fff}
:only-of-type() p:only-of-type() {color: #fff} 자식요소가 하나일때 선택자의 칼라를 흰색으로 나타냅니다.
:not() li:not(:nth-child(2)) {color: #fff} 두번째 li요소를 제외한 요소의 칼라를 흰색으로 나타냅니다.
:root :root {color: #fff} 문서 구조에서 root 요소(가장 상위 단계의 부모 요소)에 칼라를 흰색을 적용합니다.
:empty :empty {color: #fff} 자식 요소(텍스트 공백 포함)가 존재하지 않는 경우에 칼라를 흰색으로 적용합니다.

가상 요소

종류 예시 설명
::first-line p::first-line {color: #fff} 문단의 첫 줄에 컬러색을 하얀색으로 변경합니다.
::first-letter p::first-letter {color: #fff} 문단의 첫 글자에 컬러색을 하얀색으로 변경합니다.
::before p::before {content:'시작'} 문단 시작 부분에 '시작'이라는 요소를 추가합니다.
::after p::after {content:'끝'} 문단 끝 부분에 '끝'이라는 요소를 추가합니다.
::selection p::selection {color: red} 드래그로 선택하였을 경우 글씨 색을 하얀색으로 변경합니다.

링크 가상 선택자

종류 예시 설명
:link a:link {color: #fff} 방문 전 링크 상태색을 흰색으로 나타냅니다.
:visited a:visited {color: #fff} 방문 후 링크 상태색을 흰색으로 나타냅니다.
:hover a:hover {color: #fff} 마우스 오버했을 때 링크상태색을 흰색으로 나타냅니다.
:active a:active {color: #fff} 클릭했을때 링크 상태색을 흰색으로 나타냅니다.

폼 선택자

종류 예시 설명
:checked input:checked {color: #fff} 사용자가 요소를 체크했거나 선택한 경우 칼라색을 흰색으로 설정합니다.
:focus input:focus {color: #fff} 포커스를 받으면 칼라색을 흰색으로 설정합니다.
:enabled input:enabled {color: #fff} 선택, 클릭, 입력 등을 하거나 포커스를 받게되면 칼라색을 흰색으로 설정합니다.
:disabled input:disabled {color: #fff} input 태그에 disabled 가 걸려있으면, 입력창이 비활성화되며 입력이 제한되며, 색을 흰색으로 설정합니다.
